# freelance-cinematic-brand-site

シネマティックブランドサイトの環境構築。

## Stack

- Next.js 16.2.2 (App Router / TypeScript)
- Tailwind CSS 4.2.2
- GSAP 3.14.2 / @gsap/react 3.14.2
- ESLint + Prettier
- Inter (next/font)

## Included

- TypeScript strict mode
- path alias `@/`
- Tailwind design tokens via CSS variables
- GSAP + ScrollTrigger registration scaffold
- Inter font setup
- ESLint / Prettier config

## Setup

```bash
npm install
npm run dev
npm run build
npm run lint
```
