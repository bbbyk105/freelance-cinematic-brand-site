# freelance-cinematic-brand-site

シネマティックブランドサイトの環境構築。

## Stack

- Next.js App Router
- Tailwind CSS
- GSAP

## Routes prepared

- `/`
- `/services/brand-film`
- `/services/music-video`
- `/services/commercial`
- `/services/documentary`
- `/services/creative-direction`
- `/news`
- `/contact`
